# AI_Chat_Widget
Have you ever spent hours researching the specifics of a random product that you needed to buy, comparing the endless list of options that you have available to you for you to only give up? Additionally, we found current solutions to this issue like LiveChat incomplete, lacking accessibility, personalization, and general helpfulness. Our team noticed these issues and decided to create a solution to this. An innovative widget that can be added to any sales website that web scrapes the data from the website to help users find the product perfect for them. We are planning to do this using a combination of Generative AI, Accessible Interface, Proompting, and live voice calls with a professional.
